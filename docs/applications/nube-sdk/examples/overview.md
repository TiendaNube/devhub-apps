Welcome to the NubeSDK examples section! Here you'll find practical implementations to help you get started quickly with your applications.

| Example                                              | Description                                                         |

| [Carousel](./carousel)                               | Create an image carousel/slider for product showcases or promotions |
| [Banner](./banner)                                   | Display promotional banners with customizable content and styling   |
| [FAQs](./faq)                                        | Build an expandable FAQ section using the Accordion component       |
